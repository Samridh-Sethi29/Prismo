# PRISMO

## Project Overview

Prismo is a modern entertainment platform where users can discover movies, TV shows, and music, create personalized watchlists, write reviews, rate content, interact with other users, and receive intelligent recommendations.

The goal is to combine the best ideas from platforms like Netflix, Spotify, YouTube, IMDb, and Letterboxd into one seamless experience.
